# minitalk

## Description

The goal of this project is to understand how signals work in C as well a bit manipulation.

## Instructions

Use `make` to create the server and client programs.

Launch `server`, it should print its PID

Use a second terminal to send a message to the server using `client [Server PID] [Message]`
